# 📦 Order Management System

## 🚀 Features
- JWT Authentication
- Create, Update, Delete Orders
- Order Status Tracking
- Protected Routes

## 🛠 Tech Stack
- React
- Node.js
- Express
- MongoDB
- Tailwind CSS

## 📁 Project Structure
frontend/ → React App  
backend/ → Node API  

## ⚙️ Setup
### Backend
cd backend
npm install
npm run dev

### Frontend
cd frontend
npm install
npm run dev
